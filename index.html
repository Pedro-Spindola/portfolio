To-do List

Este projeto foi criado com o objetivo de desenvolver uma To-do List simples e fácil de usar. Utilizei HTML, CSS e JavaScript para a criação da interface e para a lógica da aplicação. A To-do List conta com funcionalidades básicas, como adicionar tarefas, marcar como concluídas e excluir tarefas. O maior desafio enfrentado foi implementar a função de armazenamento local, para que as tarefas fossem salvas mesmo após o fechamento do navegador. No final, fiquei satisfeito com o resultado e aprendi muito durante o processo de desenvolvimento.
